## laravel-vue-role-based

a simple laravel vue application that has user rules Admin And Users

### setup project

```
$ npm install
$ composer install
$ cp .env.example .env 
```

### setup environment
open file .env
```env
DB_DATABASE=YOUR_DATABASE_NAME
DB_USERNAME=YOUR_DATABASE_USERNAME
DB_PASSWORD=YOUR_DATABASE_PASSWORD
```

### run project
```
$ npm run watch
$ php artisan serve
```
